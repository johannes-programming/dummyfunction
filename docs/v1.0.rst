Introduction
------------

This project provides the dummyfunction.

Installation
------------

To install dummyfunction, you can use pip. Open your terminal and run:

.. code-block:: shell

    pip install dummyfunction

--------

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

~~~~~~~~~~~~~~~~~~~~~~

``dummyfunction.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
